# jupyterlab_practice
## Notebooks
- question11.ipynb Implements a function to compute the mean of a list. Includes an explanation in Markdown with Latex equations
- question12.ipynb shows using shell commands in Jupyter and timing Python operations with `timeit` with the directions of question 12
## Quick Start
1. Clone the repository
2. Creat a Conda environment using the included .yaml file
3. Activate new environment to copy dependencies
4. Navigate to the cloned directory
5. Launch JupyterLab and open the notebooks
